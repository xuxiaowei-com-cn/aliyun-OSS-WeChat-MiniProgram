# 使用方法

- 修改 /aliyunoss/upload.js 中的 accessid、accesskey、host

- 修改 project.config.json 中的 appid 为你的 appid

# 工具未校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书。
